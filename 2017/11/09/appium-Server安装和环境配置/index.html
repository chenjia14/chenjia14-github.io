appium Server安装和环境配置

发表于 2017-11-09 | 分类于 自动化测试

安装java

1. 下载JDK

JDK8下载地址

2. 配置java环境变量

新建 JAVA_HOME

C:\Program Files\Java\jdk1.8.0_151

新建 CLASSPATH

.;%Java_Home%\bin;%Java_Home%\lib\dt.jar;%Java_Home%\lib\tools.jar

编辑 Path （编辑文本并在最前面增加）

%Java_Home%\bin;%Java_Home%\jre\bin;

通过java -version命令验证

安装 Android SDK

1. UI方式安装（windows推荐）

下载Android SDK Tools并进行安装，安装目录推荐为C:\Program Files\android-sdk。安装默认勾选的package即可

记得创建临时文件夹temp并以管理员模式运行SDK Manager
http://www.androiddevtools.cn/

2. 命令行方式安装

虽然 Android Studio 给我们提供了可视化页面来管理 SDK，但还是会有在命令行管理 SDK 的需求，在 SDK tools 25.2.3 版本以前 SDK 管理命令是 android，从 25.2.3 开始就提供了 sdkmanager 命令专门来管理 SDK

SDK tools下载地址，使用说明见 官方 sdkmanager 文档

列出所有已安装、可安装、可更新的包

sdkmanager –list

根据列出内容选择安装

sdkmanager “platforms;android-27”

双引号里面的就是Path那一列显示的值，同理要安装其他的包就使用对应的Path

报错信息 repositories.cfg could not be loaded

手动添加 repositories.cfg

其他的使用方法见 http://www.jianshu.com/p/06f1a37de2a6

3. 设置环境变量

新增 ANDROID_HOME

C:\Program Files\android-sdk

编辑 Path （编辑文本并在最前面增加）

;%ANDROID_HOME%\platform-tools;%ANDROID_HOME%\tools

通过adb version命令验证

安装 appium Server

安装说明 http://www.testclass.net/appium/appium-base-server/

记得添加到环境变量path以允许appium-doctor：
C:\Program Files\Appium\node_modules.bin

运行命令appium-doctor测试环境如下：

# appium
